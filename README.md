## 📖 About This Tool

Disentangling CPT helps you update language models. It uses special routing methods to keep core knowledge intact while the model learns new facts. Many language models lose their original skills when they learn new tasks. This tool prevents that loss. It ensures the model stays smart in its original areas while it adopts new information.

You can use this software if you want to adapt a model for a specific topic or field. It manages the training process so the model remains stable and accurate. You do not need deep math knowledge to use this. You provide the model and the data, and the software handles the internal adjustments.

## 🛠️ Using the Software

When you open the software, you see a main menu. This menu guides you through the process of adapting your model.

### Preparing Your Data
The software needs text files for its training process. Place your text documents in a folder that you can easily reach. Ensure these files consist of plain text. The software reads these files to understand the new information you want to teach the model.

### Starting the Training
1. Click the "Load Model" button at the top of the screen.
2. Select the existing model you want to improve.
3. Click the "Add Data" button and choose the folder containing your text files.
4. Review the settings. The software chooses the best settings by default. Most users do not need to change these.
5. Click "Begin Training."
6. The software displays a progress bar. Wait for this bar to reach 100 percent. The time this takes depends on how much text you provided.

## 📊 Understanding the Output

Once the process finishes, the software creates a new folder. This folder contains your updated language model. You can now use this model in other applications that support standard model formats.

If you see an error message, check the log file located in the same folder as the program. The log file explains any issues that stopped the training. Common issues include a lack of free disk space or unsupported file formats.

## ⚙️ Advanced Settings

You can adjust how the software performs its work. Open the "Settings" menu to view these options.

*   **Routing Intensity:** This controls how much the model changes during training. Higher values mean more changes. Keep this at the default level if you are unsure.
*   **Hub Preservation:** This toggle keeps the core knowledge segments protected. You should keep this turned on for better stability.
*   **Training Speed:** This adjusts how fast the software completes the tasks. Higher speeds use more of your computer’s processor.

## 💡 Troubleshooting Tips

*   **Software will not start.** Restart your computer and try again. Ensure your graphics card drivers are current.
*   **Training stops midway.** Check that your computer has enough space for the output files. Large models create large files.
*   **The model performance is poor.** Check if your text files contain clear and accurate information. The software learns from whatever data you provide. Poor data leads to poor results.
*   **The program freezes.** Allow the computer time to process the information. Training language models requires significant power. Avoid opening other heavy applications while the software runs.
